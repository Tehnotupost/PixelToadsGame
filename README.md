# PixelToadsGame

PixelToadsGame 🐸
PixelToadsGame — это пиксельный карточный рогалик про жаб.
🎮 Текущий статус
✅ Хаб
✅ Механика кормления (клик по мухам)
⏳ Карточная система
⏳ Бои

🕹 Скачать и поиграть
Скачать последнюю версию можно здесь:
👉 https://github.com/Tehnotupost/PixelToadsGame/releases

### Как запустить:
1. Установить Java (если нет)
2. Скачать .jar
3. Запустить:

java -jar PixelToads-1.0.0.jar
